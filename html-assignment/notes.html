HTML tags

Paragraphs
A paragraph tag is a paired tag which presents paragraphs for containing bodies of texts. The paragraph tag creates spaces in-between that separates it from other HTML tags.
Body
The body tag in HTML is a paired tag that is used to present the main body content inside an HTML page.
Header
A header is a paired tag that is used to present webpage introductory or navigational contents
Headings
Headings are used to present the headings on a webpage. They are used to differentiate importants from the rest of the content, usually having strong characteristic. Headings consists of 6 elements which are h1, h2, h3, h4, h5, and h6.
Anchor
An anchor tag is used to create a hyperlink in a webpage. It is a paired tag that has href attribute to provide link location.
